**最終更新**: 2026-05-05
**バージョン**: draft-v0.1.0 (Phase 10.B)

## 1. はじめに

本プライバシーポリシー (以下「本ポリシー」という) は、運営者 ([TBD: Phase 10.D 後に氏名 / ハンドルを記載]) が個人運営する画像注釈共有サービス snap-share (以下「本サービス」という) における、ユーザーから取得する情報の種類、利用目的、第三者提供、保存期間、開示請求の手続きについて定めます。

本サービスを利用する場合、ユーザーは本ポリシーの内容に同意したものとみなされます。

## 2. 個人情報を取得する基本方針

本サービスは、**アカウント登録不要・cookieless** で利用できる軽量サービスとして設計されています。氏名 / メールアドレス / 電話番号などの直接的な個人情報は原則として取得しません。

## 3. 取得する情報

本サービスは、以下の情報を自動的に取得・処理します。

| 情報 | 取得タイミング | 保存場所 | 保存期間 |
|------|----------------|----------|----------|
| アップロードされた画像 (PNG / JPEG / WebP / SVG) | ルーム作成時 | Cloudflare R2 (米国 / 欧州リージョン) | TTL に従う (24 時間〜最大 7 日) |
| 注釈データ (矩形 / 矢印 / テキスト / ハイライトの座標・色・テキスト等) | 注釈作成・編集時 | Cloudflare Durable Object | TTL に従う |
| ルームパスワードのハッシュ (PBKDF2-SHA256 210k iter) | パスワード保護時 | Cloudflare R2 (メタデータと同じ場所) | TTL に従う |
| アクセス IP アドレス | API リクエスト時 | Cloudflare ログ (短期間) / Workers ログ | Cloudflare の保持ポリシーに従う (通常数日) |
| User-Agent | API リクエスト時 | 上記同様 | 上記同様 |
| アクセス日時・経路情報 | API リクエスト時 | 上記同様 | 上記同様 |
| Web Analytics データ (ページビュー、リファラ等の集計値) | ページアクセス時 | Cloudflare Web Analytics (cookieless) | Cloudflare の保持ポリシーに従う |

**重要**:

- ユーザーは本サービスにアップロードする情報を自身で管理する責任を負います。**個人情報・機密情報をアップロードしないでください。**
- 画像と注釈データはルームの TTL 期限経過後、自動的に削除されます。重要なデータは TTL 期限内に PNG エクスポート等で別途保全してください。

## 4. 利用目的

取得した情報は、以下の目的のために利用します。

1. 本サービスの提供 (画像 / 注釈の保存・配信、共同編集機能の動作)
2. 本サービスの運営・改善 (バグ修正、機能追加の判断材料、UI / UX 改善)
3. 不正利用の検知・防止 (Cloudflare Turnstile、Workers Rate Limit、SHA-256 ブラックリスト等)
4. 利用統計の作成・分析 (Cloudflare Web Analytics による集計データの解析)
5. 通報対応 (第 4 条違反コンテンツの確認・対応)
6. 法令に基づく対応 (司法当局からの正式な要請等)

## 5. Cookie / トラッキング技術

1. 本サービスは、追跡目的の Cookie を使用しません (cookieless 方針)。
2. ブラウザの sessionStorage / localStorage を、ローカル状態の保持目的でのみ使用します (例: ルームアクセス用 token の一時保存)。これらは本サービス外に送信されません。
3. アクセス解析には Cloudflare Web Analytics (cookieless 設計、IP アドレスを匿名化集計) を使用します。
4. Google Analytics / Facebook Pixel / その他第三者の広告トラッキングは現時点で導入しておりません。今後導入する場合は、本ポリシーの改定で告知します。

## 6. 第三者への提供

1. 本サービスは、ユーザーから取得した情報を第三者に販売・譲渡しません。
2. ただし、以下の場合を除き、ユーザーの同意なく情報を提供することはありません。
   - 司法当局からの正式な要請があり、これに応じる法的義務がある場合
   - 第三者の生命・身体・財産に対する重大な危険を回避するために必要な場合
   - 本サービスの運営に必要な範囲で、業務委託先 (Cloudflare 等のインフラ事業者) に提供する場合 (これらは本ポリシーと同等以上の管理水準を有することを確認しています)

## 7. 保存期間

1. **画像 / 注釈 / 認証メタデータ**: ルーム作成時に指定された TTL (デフォルト 24 時間、最大 7 日間) を経過すると自動的に削除されます。
2. **アクセスログ**: Cloudflare の保持ポリシーに従います。運営者側で長期保管することはありません。
3. **Web Analytics データ**: Cloudflare の保持ポリシーに従います。

## 8. 開示請求・削除請求・苦情

1. ユーザーは、自身に関連する情報の開示・削除・利用停止を請求できます。
2. ご請求は、以下の連絡先までお願いします。
   - **GitHub Issue** (推奨): <https://github.com/imotako-pum/snap-share/issues>
   - **メール**: [TBD: Phase 10.F 公開時に連絡先メールを記載]
3. 請求内容と本人確認方法 (本人性の確認が必要な場合) について、合理的な範囲でご協力ください。

## 9. 安全管理

1. 通信は HTTPS で暗号化します。
2. パスワード保護されたルームのパスワードは、PBKDF2-SHA256 (210,000 iter) でハッシュ化して保存します。平文の保存は行いません。
3. WebSocket 接続では、JWT を URL に乗せず、1-shot 60 秒 ticket 方式を採用しています (Cloudflare KV 経由)。
4. アップロードされた画像は SHA-256 ハッシュをベースとした blocklist で違法 / 不正コンテンツの再アップロードを抑止します。
5. 本サービスは個人運営であり、エンタープライズグレードの SLA / 監査対応は提供しません。

## 10. 子どもの利用

本サービスは年齢制限を設けていませんが、13 歳未満のユーザーは保護者の同意のもとで利用してください。13 歳未満のユーザーから取得したことが判明した個人情報は、確認次第速やかに削除します。

## 11. 国境を越えるデータ移転

本サービスは Cloudflare のグローバルインフラを利用しており、ユーザーの情報は米国 / 欧州 / アジア太平洋を含む複数の地域で処理される可能性があります。Cloudflare のデータ処理方針については [Cloudflare Privacy Policy](https://www.cloudflare.com/privacypolicy/) を参照してください。

## 12. 改定

1. 運営者は、必要に応じて本ポリシーを改定できます。
2. 改定する場合、変更内容と効力発生日を本サービス上または GitHub の CHANGELOG / リリースノートで告知します。
3. 重要な変更を行う場合は、可能な範囲でユーザーに事前告知します。
