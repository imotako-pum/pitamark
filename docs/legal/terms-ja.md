## 第 1 条 (本規約の適用)

本規約は、運営者 ([TBD: Phase 10.D 後に氏名 / ハンドルを記載]) が個人運営する画像注釈共有サービス (以下「本サービス」という) の利用条件を定めるものです。利用者 (以下「ユーザー」という) は、本サービスを利用することで、本規約のすべてに同意したものとみなされます。

## 第 2 条 (サービス内容)

1. 本サービスは、ユーザーがアップロードした画像に対して、矩形 / 矢印 / テキスト / ハイライトの注釈をオーバーレイし、共有 URL を発行する機能を提供します。
2. 共有 URL を知る複数のユーザーが、同じ画像に対してリアルタイムで共同編集できる機能を提供します。
3. アップロードされた画像と注釈データは、ルーム作成時に指定された有効期限 (デフォルト 24 時間、最大 7 日間) を経過すると自動的に削除されます。フリーミアム機能 (将来実装予定) を利用する場合のみ、有効期限を延長できる場合があります。
4. 本サービスは無償で提供されますが、運営者の判断により予告なく仕様変更・機能追加・サービス停止を行うことがあります。

## 第 3 条 (利用条件)

1. ユーザーは、自己の責任において本サービスを利用するものとします。
2. ユーザーは、適用される法令および本規約を遵守して本サービスを利用するものとします。
3. アカウント登録は不要で、いかなるユーザーも共有 URL があれば本サービスにアクセスできます。

## 第 4 条 (禁止事項)

ユーザーは、本サービスの利用にあたり、以下の行為をしてはなりません。

1. 法令に違反する行為 (著作権侵害、誹謗中傷、児童ポルノ、わいせつ物頒布、薬物の販売告知、その他犯罪に関連する行為)
2. 第三者の権利 (著作権、肖像権、プライバシー、パブリシティ権、その他一切の権利) を侵害する行為
3. 他のユーザー、運営者、または第三者に対する誹謗中傷・名誉毀損・脅迫・嫌がらせ
4. 個人情報・機密情報の無断アップロード (他者の個人情報、機関 / 組織の機密情報など)
5. 本サービスの運営を妨害する行為 (過剰なリクエスト、自動化された大量アップロード、リバースエンジニアリング、セキュリティの脆弱性を悪用する行為)
6. スパム / マルウェア / フィッシング / その他悪意あるコンテンツの配布
7. 商標 / ブランドの不正利用
8. 反社会的勢力への利益供与
9. その他、運営者が不適切と判断する行為

違反が認められた場合、運営者は事前通告なくコンテンツの削除、アクセス制限、TTL 短縮を行うことがあります。

## 第 5 条 (知的財産権)

1. ユーザーがアップロードした画像、および注釈データの著作権はユーザー (または正当な権利者) に帰属します。
2. ユーザーは、本サービスを利用することにより、本サービスの運営・改善のために必要な範囲で、運営者がこれらコンテンツを利用 (保存、表示、削除) することを許諾するものとします。
3. 本サービス自体 (ソースコード、UI、ドキュメント、ロゴ等) の知的財産権は、運営者または該当するライセンサーに帰属します。本サービスのソースコードは [MIT License](https://github.com/imotako-pum/snap-share/blob/main/LICENSE) で公開されています。

## 第 6 条 (免責事項)

1. 本サービスは「現状有姿」で提供され、運営者は本サービスの正確性、完全性、有用性、特定目的への適合性、エラー / 中断のないこと、セキュリティ等について、明示・黙示を問わず一切の保証をしません。
2. ユーザーが本サービス上で作成したデータは TTL により自動削除されます。運営者はデータの消失、誤削除、復旧不能について一切責任を負いません。重要なデータは TTL 期限内に PNG エクスポート等で別途保全してください。
3. ユーザーが本サービスを利用したことにより生じた損害 (機会損失、データ損失、逸失利益等を含む) について、運営者は、運営者の故意または重過失による場合を除き、一切の責任を負いません。
4. 本サービスは個人運営であり、エンタープライズ用途・厳格な機密管理・SLA 保証を要する用途には適しません。

## 第 7 条 (サービスの変更・中断・終了)

1. 運営者は、ユーザーへの事前通告なく、本サービスの内容を変更、追加、廃止することができます。
2. 運営者は、定期メンテナンス、緊急保守、災害、第三者の妨害行為、その他運営者がやむを得ないと判断する事由により、本サービスの全部または一部を一時的に中断することができます。
3. 運営者は、合理的な事前告知 (CHANGELOG / GitHub リポジトリ / 本サービス上の通知のいずれかの方法) を行ったうえで、本サービスを終了することができます。

## 第 8 条 (通報・違反コンテンツへの対応)

1. ユーザーは、本サービス上で第 4 条に該当するコンテンツを発見した場合、以下の方法で運営者に通報できます。
   - **GitHub Issue** (推奨): ラベル `report-abuse` を付けて issue を作成 ([テンプレート](https://github.com/imotako-pum/snap-share/issues/new?template=abuse-report.yml))
   - **メール** (緊急時): [TBD: Phase 10.F 公開時に連絡先メールを記載]
2. 通報を受領後、運営者は内容を確認し、合理的な対応 (コンテンツの削除、TTL 短縮、追加調査) を行います。ただし、すべての通報に個別回答することは保証しません。

## 第 9 条 (プライバシー)

ユーザーの個人情報 / 利用情報の取り扱いについては、別途定める [プライバシーポリシー](./privacy-ja.md) によります。

## 第 10 条 (本規約の改定)

1. 運営者は、必要に応じて本規約を改定できます。
2. 改定する場合、変更内容と効力発生日を本サービス上または GitHub の CHANGELOG / リリースノートで告知します。
3. 効力発生日以降にユーザーが本サービスを利用した場合、改定後の規約に同意したものとみなされます。

## 第 11 条 (準拠法・裁判管轄)

1. 本規約の解釈と適用は、日本法に準拠するものとします。
2. 本規約または本サービスに関連して生じた紛争については、東京地方裁判所を第一審の専属的合意管轄裁判所とします。
